Detailed Description

Defines string constraints. These are formulas talking about strings. We implemented two forms of constraints: string_constraintt are formulas of the form \(\forall univ\_var \in [lb,ub[. prem => body\), and not_contains_constraintt of the form: \(\forall x \in [lb,ub[. p(x) => \exists y \in [lb,ub[. s1[x+y] \ne s2[y]\).
