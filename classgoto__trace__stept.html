Detailed Description

Step of the trace of a GOTO program.

A step is either:

  • an assignment
  • an assume statement
  • an assertion
  • a goto instruction
  • a constraint (unused)
  • a function call
  • a function return
  • a location (unused)
  • an output
  • an input
  • a declaration
  • a dead statement
  • a shared read (unused)
  • a shared write (unused)
  • a spawn statement
  • a memory barrier
  • an atomic begin (unused)
  • an atomic end (unused)
